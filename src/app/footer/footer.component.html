About

WinIsen est un site web qui permet de suivre les résultats des championnats de football.
Il a été réalisé dans le cadre du projet d'Angular de la formation Ingénieur de l'ISEN Toulon.
Il est développé en Angular 11 et utilise l'API Football Data.
Le code source est disponible sur GitHub.
